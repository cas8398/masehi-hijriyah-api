 # Hijri Calender
mengkonversi tanggal masehi ke hijriyah

# Demo dan Penggunaan
https://rapidapi.com/cas8398/api/masehi-ke-hijriyah 
```json
Pilih tanggal dengan format : 02/05/2023
Perlu diperhatikan pada tanggalnya : 
tanggal : 2 angka
bulan : 2 angka
tahun : 4 angka 
```
